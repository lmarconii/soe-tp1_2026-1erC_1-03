- En este primer trabajo práctico, exploramos las funciones básicas de creación, eliminación y asignación/cambio de prioridad a tareas de FreeRTOS, analizando un proyecto que implementa la interacción básica entre un botón y un led.

- Más adelante, el proyecto se modifica para generalizar la interacción a múltiples botones y leds mediante la instanciación múltiple de una misma tarea. También se implementa el procesamiento periódico de las tareas mediante la primitiva `vTaskDelayUntil`.
